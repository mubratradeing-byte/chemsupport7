## 🔬 ඔර්ගනික් රසායනවිද්‍යා (Organic Chemistry)

### 1. ස්ටර්‍යෙල්‍ය නිරඨමතිකතාවය (Stereoisomerism)

**ප්‍රශ්ནය:**
```
$CH_3CH(OH)CH_2CH_3$ වල stereo-isomers නොමැතිවිසින් පිළිතුරු දෙන්න. 
Absolute configuration සහිතව විස්තර කරන්න.
```

**ප්‍රශ්නය:**
```
Thalidomide වල stereoisomers සම්බන්ධයෙන් එහි biological effects ගැන කතා කරන්න.
```

---

### 2. අभିකර්ම ක්‍රම (Reaction Mechanisms)

**ප්‍රශ්නය:**
```
$SN_2$ මෙකানිසම් සඳහා නිර්දේශන සහ අනුක්‍රමිකය දෙන්න. 
$SN_1$ සම්පූර්ණ වටපිටාවලට තුලනය බිම.
```

**ප්‍රශ්නය:**
```
Grignard reagent ($RMgX$) නිර්මාණ ක්‍රම සහ එහි ප්‍රතිබන්ධතා විස්තර කරන්න.
```

### 3. අර්ධ-අंశ සමිතික ක්‍රයා (Electrophilic Addition)

**ප්‍රශ්නය:**
```
$C=C$ බන්ධනයට $HBr$ එකතු වීමේ ක්‍රමය දේখා Markovnikov's rule අදාල කරන්න.
```

**ප්‍රශ්නය:**
```
Benzene رينگ ඉතිරි ක්‍රයා සඳහා directing effects සහ activating/deactivating groups 
පිළිබඳ විස්තරිත පරිවර්තනයක් දෙන්න.
```

## ⚛️ අකර්ගනික් රසායනවිද්‍යා (Inorganic Chemistry)

### 1. සම්බන්ධන වර්ගයන් (Bonding Types)

**ප්‍රශ්නය:**
```
$NH_3$ සහ $PCl_3$ වල බන්ධන ස්වභාවය අවසාබිතු කරන්න. 
Hybridization සහ geometry විස්තර කරන්න.
```

**ප්‍රශ්නය:**
```
Ionic සහ Covalent බන්ධන අතර වෙනස්කම් විස්තරිතවම පෙන්වන්න. 
উদාහරණ ද දෙන්න.
```

---

### 2. සම්පර්කීය සංයෝගයන් (Coordination Compounds)

**ප්‍රශ්නය:**
```
$[Cu(NH_3)_4]^{2+}$ නිපදවා crystal field theory එක්‍රීතව දෙන්න.
Splitting pattern සහ magnetic properties විස්තර කරන්න.
```

**ප්‍රශ්නය:**
```
Isomerism සම්පර්කීය සංයෝගයන්හි: Structural සහ stereoisomers සඳහා 
$[Co(NH_3)_5(SCN)]^{2+}$ උදාහරණ ශ්‍රි ලන්කා.
```

---

### 3. Redox ප්‍රතිකර්යා (Redox Reactions)

**ප්‍රශ්නය:**
```
Half-reaction method භාවිතයෙන් සම්පූර්ණ කරන්න:
$MnO_4^- + C_2O_4^{2-} \rightarrow Mn^{2+} + CO_2$ (අම්ල මාධ්‍යයේ)
```

**ප්‍රශ්නය:**
```
Electrochemistry සඳහා: Standard electrode potentials භාවිතයෙන් 
cell potential ගණනය කරන ක්‍රම දෙන්න.
```

## 🌡️ භෞතික ගුණ රසායනවිද්‍යා (Physical Chemistry)

### 1. Thermodynamics

**ප්‍රශ්නය:**
```
$ΔG = ΔH - TΔS$ සඳහා සිංහල ප්‍රකාශන: 
Gibbs free energy Spontaneity සිතට පිළිතුරු දෙන්න සම්පූර්ණ ගණිතමය වසර.
```

**ප්‍රශ්ნය:**
```
$\Delta H_f$ සහ $\Delta H_c$ දෙස සිතින්න. Hess's Law භාවිතයෙන් 
සිંහල මාධ්‍යෙන් උදාහරණ ගැලපුණු කරන්න.
```

---

### 2. Kinetics (ගතිශීලිතාව)

**ප්‍රශ්නය:**
```
Rate equation සඳහා: Zero order, First order සහ Second order ප්‍රතිකර්යා 
සඳහා අරමුණු නිර්ණයක්‍ර අවසාබිතු කරන්න (සිංහල).
```

**ප්‍රශ්නය:**
```
Activation energy සහ Arrhenius equation පිළිබඳ: 
$k = Ae^{-E_a/RT}$ සඳහා සිංහල විස්තරණයක් දෙන්න.
```

---

### 3. Equilibrium (සමතුලිතობ)

**ප්‍රශ්නය:**
```
$N_2 + 3H_2 \rightleftharpoons 2NH_3$ සඳහා $K_c$ සහ $K_p$ 
අතර සම්බන්ධතාවය සිංහල මාධ්‍යෙන් පෙන්වන්න.
```

**ප්‍රශ්නය:**
```
Le Chatelier's principle භාවිතයෙන්: Temperature, Pressure සහ 
Concentration වෙනස් කිරීම equilibrium ගතිවිද්‍යාව පිළිබඳ 
විස්තරිත පිළිතුරු දෙන්න.
```

---

### 4. Electrochemistry (විද්‍යුත් රසායනවිද්‍යා)

**ප්‍රශ්නය:**
```
Galvanic/Voltaic cell සඳහා: Anode, Cathode සහ electron flow 
පිළිබඳ සිංහල පැහැදිලි කිරීම් දෙන්න. $E°_{cell} = E°_{cathode} - E°_{anode}$ 
භාවිතයෙන්.
```

**ප්‍රශ්නය:**
```
Electrolysis මාධ්‍යවල: Faraday's laws සහ ප්‍රතිකර්යා සිංහල 
ගණිතමය සිතකින් විස්තරිතව පෙන්වන්න.
```
